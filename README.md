## 💡 주요 기능
- **상품 상세페이지** : 원하는 상품을 선택할 때, 상세 페이지를 통해 더욱 자세하게 상품을 구경할 수 있습니다.
- **장바구니** : 원하는 상품을 상세 페이지에서 추가 시, 장바구니 페이지에서 담은 상품 목록과 총금액을 확인할 수 있습니다.
- **최근 본 상품** : 오른쪽 사이드에 존재하는 박스에서 최대 2개까지 최근 본 상품을 확인할 수 있습니다.
